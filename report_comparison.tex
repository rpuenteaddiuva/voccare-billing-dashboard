\section{Definicion de la Nueva Calculadora (v2.1)}

La nueva pol\'itica de facturaci\'on busca incentivar la adopci\'on digital y reflejar los costos reales de operaci\'on.

\begin{itemize}
    \item \textbf{Fee Mensual:} \$3,150 USD.
    \item \textbf{Servicios Concluidos (SC):} Se suman todos los servicios (Voz + App) para determinar el Tier.
    \item \textbf{Incentivo App:} Se aplica un \textbf{descuento del 10\%} sobre el costo proporcional de los SC originados por App.
    \item \textbf{Transacci\'on Digital:} \$0.45 USD por servicio App exitoso.
    \item \textbf{Llamadas V\'alidas:} Se cobran seg\'un ocurrencia real, ajustadas por un factor de eficiencia t\'ecnica por pa\'is.
\end{itemize}

\subsection*{Formulas de Calculo Exactas}

Para una total transparencia, a continuacion se presentan las formulas exactas utilizadas para calcular la facturacion bajo ambas politicas:

\textbf{Variables Clave:}
\begin{itemize}
    \item $F_{base}$: Fee Mensual ($3150 \text{ USD}$).
    \item $C(Vol, Tiers, A)$: Funcion que calcula el costo para un volumen dado, aplicando una estructura de Tiers ($Tiers$) y un factor de ajuste de precios ($A$).
    \item $V_{SC}$: Volumen Total de Servicios Concluidos (App + Voz).
    \item $V_{APP}$: Volumen de Servicios Concluidos via App.
    \item $R_{LV/SC}$: Ratio de Llamadas Validias por Servicio Concluido (ej. $1.02$ para Dominicana, $3.00$ para Puerto Rico).
    \item $A$: Factor de Ajuste de Precios (normalmente $1.0$, se aplica para alinear con tarifas especificas de contrato).\
    \item $D_{APP} = 0.10$: Descuento del 10\% para servicios App.
\end{itemize}

\textbf{1. Ecuacion para la Politica 2024 (Factura Antigua):}
\[
\text{Factura}_{2024} = (F_{base} \times A) + C(V_{SC}, T_{SC}, A) + C(V_{SC} \times R_{LV/SC}, T_{LV}, A)
\]

\textbf{2. Ecuacion para la Politica 2025 (Factura Nueva con Incentivo):}
\begin{enumerate}
    \item \textbf{Costo Base de SC:}
    \[ Costo_{SC\_Base} = C(V_{SC}, T_{SC}, A) \]
    \item \textbf{Descuento por Uso de App en SC:}
    \[ Descuento_{SC\_App} = Costo_{SC\_Base} \times \left(\frac{V_{APP}}{V_{SC}}\right) \times D_{APP} \quad (\text{si } V_{SC} > 0 \text{, si no } 0) \]
    \item \textbf{Costo Neto de SC (con Descuento):}
    \[ Costo_{SC\_Neto} = Costo_{SC\_Base} - Descuento_{SC\_App} \]
\end{enumerate}
La factura total de 2025 es:
\[
\text{Factura}_{2025} = (F_{base} \times A) + Costo_{SC\_Neto} + C(V_{SC} \times R_{LV/SC}, T_{LV}, A) + C(V_{APP}, T_{APP}, A)
\]
